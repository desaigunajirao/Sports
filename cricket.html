Rules/Facts about Cricket

  1. Objective:
    The primary objective in cricket is to score more runs than the opposing team, either by batting or by dismissing all the opposition's batsmen
  2. Innings:
    A cricket match consists of two innings â€“ one for each team. The team batting first aims to set a high score, while the second team tries to chase it down.
  3. Overs:
    A standard limited-overs match consists of a set number of overs (usually 20 or 50), where each bowler is allowed to bowl a maximum of 6 balls per over.
  4. Wickets:
    A team loses a wicket each time a batsman is dismissed. Common ways to be out include being bowled, caught, run out, or leg before wicket (LBW).
  5. Run Out:
    A batsman can be run out if the fielding team successfully breaks the stumps with the ball in hand while the batsman is out of his crease.
